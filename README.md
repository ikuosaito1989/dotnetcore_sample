# dotnetcore_sample

> Visual Studioを使わないC#開発のリポジトリ

## Build Setup

``` bash
# install dependencies
$ dotnet restore

# serve with hot reload at localhost:5001
$ dotnet run

# watch the project
$ dotnet watch run

```
